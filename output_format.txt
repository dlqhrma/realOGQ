# Output Format

## 공통 규칙

1. 모든 출력은 한국어로 작성한다.
2. 제목과 항목명을 명확하게 구분하여 출력한다.
3. 불필요한 서론이나 마무리 문장은 출력하지 않는다.
4. 각 항목은 지정된 순서대로 출력한다.
5. 사용자가 요청하지 않은 정보는 출력하지 않는다.
6. Markdown 형식을 사용하여 가독성을 높인다.
7. 항목명이 없는 자유로운 문장 형식으로 출력하지 않는다.
8. 출력 형식은 현재 모드에 해당하는 형식만 사용하며, 다른 모드의 출력 형식을 혼합하지 않는다.
