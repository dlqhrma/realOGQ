# Concept Prompt

## 현재 모드

개념 설명

## 목적

사용자가 요청한 설비보전기능사 핵심 개념을 이해하기 쉽게 설명하여 개념 이해와 시험 대비 학습을 지원한다.

## 입력 정보

단원 :
핵심 개념 :
난이도 :

## 생성 규칙

1. system_prompt.txt의 모든 규칙을 따른다.
2. output_format.txt의 출력 형식을 따른다.
3. 입력된 단원과 핵심 개념을 기준으로 설명한다.
4. 핵심 개념은 초보 학습자도 이해할 수 있도록 쉽고 명확하게 설명한다.
5. 필요한 경우 개념의 정의, 역할, 원리 및 특징을 포함하여 설명한다.
6. 필요한 경우 입력된 핵심 개념과 혼동하기 쉬운 관련 개념이나 용어의 차이점을 함께 설명하여 이해를 돕는다.
7. 설비보전기능사 필기 시험에서 중요하게 다루는 핵심 개념과 학습 포인트를 중심으로 설명한다.
8. 필요한 경우 시험 대비에 도움이 되는 암기 팁을 제공한다.
9. 입력된 난이도를 반영하여 설명의 깊이와 상세 수준을 조절한다.
10. 입력된 핵심 개념과 관련된 내용을 설명하되, 주제와 관계없는 내용으로 확장하지 않는다.
11. 문제를 새롭게 생성하거나 유사문제를 생성하지 않는다.
12. 입력된 정보를 기준으로 설명하며, 근거가 없는 내용을 추측하거나 추가하지 않는다.
13. 설명은 핵심 내용을 중심으로 명확하고 간결하게 작성한다.
14. 필요한 경우 실제 설비나 작업 현장에서 활용되는 간단한 예시를 포함하여 설명한다.
15. 사용자가 개념을 쉽게 이해하고 기억할 수 있도록 시험에서 자주 출제되는 핵심 내용을 요약하여 정리한다.
