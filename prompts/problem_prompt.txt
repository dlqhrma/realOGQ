# Problem Prompt

## 현재 모드
문제 생성

## 목적
사용자가 선택한 조건에 맞는 설비보전기능사 CBT 객관식 문제를 생성한다.

## 사용자 입력

단원 :
난이도 :
문제 수 :

## 생성 규칙

사용자 입력값(단원, 난이도, 문제 수)을 기준으로 아래 규칙을 적용하여 문제를 생성한다.

1. system_prompt.txt의 모든 규칙을 따른다.
2. output_format.txt의 출력 형식을 따른다.
3. 사용자가 선택한 단원에서만 문제를 생성한다.
4. 사용자가 선택한 난이도를 반영한다.
5. 사용자가 요청한 문제 수만큼 생성한다.
6. 객관식(4지선다) 문제만 생성한다.
7. 생성하는 모든 문제에는 다음 정보를 함께 생성한다.

- 문제
- 보기
- 정답
- 해설
- 단원
- 핵심 개념
- 난이도

이 정보는 AI 해설, 오답노트, 유사문제 생성 및 학습분석 기능에서 동일하게 활용할 수 있도록 한다.


8. 문제와 보기는 먼저 사용자에게 표시한다. 정답과 해설은 함께 생성하되, 사용자가 각 문제의 답안을 선택하여 제출한 후에만 제공한다.
9. 동일한 요청에서 생성되는 문제는 서로 다른 개념, 상황 또는 표현을 사용하며, 동일하거나 유사한 문제가 반복되지 않도록 생성한다.
10. 요청한 문제 수를 충족하되, 동일한 문제를 반복하거나 보기만 변경한 문제는 생성하지 않는다.
11. 생성된 모든 문제는 문제 번호 순서대로 출력하며, 각 문제는 output_format.txt의 형식을 동일하게 따른다.
