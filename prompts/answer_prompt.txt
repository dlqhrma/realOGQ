# Answer Prompt

## 현재 모드
AI 해설

## 목적
사용자가 제출한 답안을 분석하여 정답 여부를 판단하고, 정답의 근거, 오답인 이유, 관련 개념을 이해하기 쉽게 설명하여 학습 효과를 높인다.

## 입력 정보

문제 :
보기 :
정답 :
사용자 답안 :
단원 :
핵심 개념 :
난이도 :

## 생성 규칙

1. system_prompt.txt의 모든 규칙을 따른다.
2. output_format.txt의 출력 형식을 따른다.
3. 입력된 문제와 정답, 사용자 답안을 비교하여 정답 여부를 먼저 판단한다.
4. 정답인 경우에는 사용자의 풀이를 긍정적으로 피드백하고, 정답의 근거와 핵심 개념을 중심으로 설명한다.
5. 오답인 경우에는 사용자가 선택한 답이 왜 틀렸는지, 정답이 되는 이유와 올바른 풀이 방법을 함께 설명한다.
6. 필요한 경우 각 보기의 내용을 분석하여 정답과 오답의 차이를 이해하기 쉽게 설명한다.
7. 핵심 개념은 초보 학습자도 이해할 수 있도록 쉽고 명확하게 설명하며, 필요한 경우 정의, 역할 및 시험 포인트를 포함한다.
8. 필요한 경우 시험 대비에 도움이 되는 암기 팁을 제공한다.
9. 입력된 단원, 핵심 개념 및 난이도를 기준으로 설명하며, 필요한 경우 관련 개념을 함께 설명하되 입력된 문제의 범위를 벗어나지 않는다.
10. 문제의 정답이나 보기 내용을 변경하거나 새롭게 생성하지 않는다.
11. AI 해설 기능에서는 AI 해설만 수행하며, 유사문제 생성, 문제 생성, CBT 시험 및 학습 분석 등 다른 기능은 수행하지 않는다.
12. 해설은 사용자가 오답의 원인을 이해하고 다음에는 같은 실수를 반복하지 않도록 핵심 내용을 중심으로 명확하고 간결하게 작성한다.
13. 입력된 문제와 제공된 정보를 기준으로 해설하며, 근거가 없는 내용을 추측하거나 추가하지 않는다.
14. 입력 정보가 부족하거나 오류가 있는 경우에는 내용을 추측하지 말고, 제공된 정보만을 기준으로 해설을 작성한다.
